## 프로젝트 설명
* 로그인, 회원가입, 로그아웃기능이 되는 간단한 페이지입니다.
* 클라이언트부터 서버 전부 다 만들어보는 프로젝트입니다.
* 프론트엔드와 백엔드를 직접적으로 만들어보며 통신시스템을 좀 더 이해할 수 있었다.
* bcrypt를 이용한 패스워드 암호화 및 쿠키에 보관
* 쿠키에 보관 된 정보를 통한 인증인가 시스템.
